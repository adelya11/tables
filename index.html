Form tags
Tags Description
<input> type="text" Setting type to "text" creates a single row field for text input
<input> type="password" Setting type to "password" creates a single row field that censors text input
<input> type="number" Setting type to "number" creates a single row field for number input
<input> type="number" Setting type to "range" creates a slider to select from a range of numbers.
<input> type="range" Setting type to "checkbox" creates a single checkbox which can be paired with other checkboxes
<input> type="radio" Setting type to "radio" creates a radio button that can be paired with other radio buttons
<input> type="list" Setting type to "list" will pair the <input> with a <datalist> element</datalist>
<input> type="submit" Setting type to "submit" creates a submit button
